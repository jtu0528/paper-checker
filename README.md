學術論文文獻智慧校核系統
本專案是一款專為學術論文寫作與文獻複查設計的 自動化文獻智慧校核系統。系統完美結合了 大型語言模型 (LLM) 的語意結構化洗滌能力 與 全球官方學術元數據中央資料庫 (Crossref API)，打造出具備高效能、低延遲、高準確度的文獻查驗方案。

透過本系統，使用者可以一鍵解決傳統論文末頁文獻（References）格式錯亂、排版換行被強行切斷、缺少作者或年份等非結構化資料的校核痛點，並實現跨平台多網域的官方文獻自動識別。

🚀 核心特色功能 (Core Features)
Gemini 智慧語意粗洗：採用 Gemini 3 Flash 作為底層 NLP 引擎，具備強大的非結構化資料提取能力，能完美縫合因 PDF 換行造成的標題碎片，100% 提取出精準的純論文標題。

Crossref 官方數據直連：捨棄易被鎖 IP 的傳統網頁爬蟲，改走官方學術綠色隧道，不需 API 金鑰、免除驗證碼卡死風險，資訊檢索速度達到毫秒級。

多網域實體特徵自動判定：不依賴模型盲猜，Java 核心透過審查回傳的 publisher 欄位與學術唯一的 DOI 識別碼前綴，100% 精準自動亮起 IEEE、ACM、Springer、Elsevier 等各大頂級期刊徽章。

Exception-First 異常置頂高亮：內建智慧排序演算法，查驗完成後會自動將「查驗失敗」或「論文不存在」的爭議文獻排在最上方並染紅高亮，方便研究人員第一時間進行手動核對。

🛠️ 正在處理的技術挑戰
高難度非結構化資料洗滌 (Advanced Unstructured Parsing)：
針對部分極度極端的論文參考文獻字串（如嚴重缺漏資訊、只有部分關鍵字），系統正持續微調 Gemini API 的 Prompt 結構。透過啟用強制 JSON 輸出規範（responseMimeType: "application/json"），已成功將大模型在邊際情況下的「幻覺 (Hallucination)」降至最低，確保各式複雜標題皆能穩定解析。

高維度關鍵字權重碰撞優化 (False Positive Filtering)：
在最新一輪的極端壓力測試中（使用非存在的自創學術標題進行測試），由於 Crossref 接口底層採用的模糊評分機制，會對高頻資工詞彙（如 Spring Boot、Real-Time）進行相似度瞎推薦。專案團隊正在整合二次驗證閘門，阻斷此類偽陽性現象。
